# Research

This directory contains research on agent memory.

## Structure

- `literature/` — Summaries of existing papers and resources
- `findings/` — Original research by contributors
- `open-questions.md` — Unsolved problems to explore

## Contributing

Add literature summaries as `literature/[topic]-[author].md`.  
Add original findings as `findings/[date]-[topic].md`.  
Update `open-questions.md` with new questions via PR.
